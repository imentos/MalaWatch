# MalaWatch 日本語 App Store 提出用ドキュメント

このファイルは App Store Connect の日本語ローカライズに使う文案です。現在の提出方針は、中国語版と同じく「IAP なし、全機能無料、Apple Watch も無料で含む」です。

- App 種類：数珠 / マーラー / 真言 / 瞑想カウンター
- 価格：無料
- In-App Purchases：なし
- アカウント登録：なし
- ネットワーク：不要。主要機能はオフラインで利用可能
- Apple Watch：無料で含まれる機能。課金対象ではない
- 珠のスタイル：すべて無料で含まれる
- プライバシー：データ収集なし、トラッキングなし、第三者 SDK なし

### 説明

```text
本物の数珠を手で送る感覚に近い、静かな数珠カウンター。

MalaWatch は、3D 数珠ホイール、オン・マ・ニ・ペ・メ・フンの六音節ガイド、音声サポート、触覚フィードバックを組み合わせた、iPhone と Apple Watch のための数珠 / マーラー / 祈りのカウンターです。真言、念仏、瞑想、呼吸法、日々の実践を落ち着いて数えられます。

一珠ずつ感じる
3D の珠は指の動きに合わせて動きます。下にスワイプすると一珠を数え、細かな触覚フィードバックでリズムを保てます。27、54、108 珠を一周の目標として選択でき、一周が終わると完了のフィードバックがあります。

六音節の真言ガイド
画面にはオン、マ、ニ、ペ、メ、フンが順番に表示されます。音声ガイドでは各音節を読み上げ、無音モードでは視覚ガイドだけを残せます。静かな場所での実践にも向いています。

周回を記録
現在の珠数と完了した周回数を見やすく表示します。進捗は端末内に保存されるため、App を閉じても続きから再開できます。

手首で静かに数える
Apple Watch 版ではタップと Digital Crown でカウントできます。一珠ごとに触覚フィードバックがあり、通勤、散歩、瞑想中、iPhone を手に取りにくい場面でも使えます。

数珠のスタイルを選択
Basic Wood、Rosewood、Walnut、Agarwood、Jade、Obsidian、Graphite の 7 種類の珠スタイルを用意しています。現在のすべてのスタイルは追加購入なしで利用できます。

オフライン、プライベート、アカウント不要
登録、ログイン、ネットワーク接続は不要です。実践データは自分の端末内に保存されます。

さまざまな実践に
仏教の真言、念仏、ヒンドゥー教の japa、カトリックのロザリオ、呼吸法、アファメーション、個人の日課など、MalaWatch は数を保ち、注意を実践そのものへ戻すための道具です。

利用規約：https://imentos.github.io/MalaWatch/terms.html
プライバシーポリシー：https://imentos.github.io/MalaWatch/privacy-policy.html
```

## App Review Notes - 日本語バックアップ

> App Review Notes は英語で貼るのがおすすめです。以下は内部確認用の日本語です。

```text
このバージョンではアカウント登録やログインは不要です。

このビルドには In-App Purchases はありません。Apple Watch 対応と珠のスタイルを含む現在のすべての機能は、追加購入なしで利用できます。

MalaWatch は iPhone と Apple Watch に対応したオフラインの数珠 / 祈り / マントラカウンターです。主な操作は、App を開く、3D 数珠ホイールを下にスワイプして数える、オン・マ・ニ・ペ・メ・フンの音節ガイドに従う、27/54/108 の目標を切り替える、珠のスタイルを選ぶ、音声ガイド / 無音モードを切り替える、現在の周回をリセットする、です。

MalaWatch はユーザーデータを収集せず、第三者 SDK を使用せず、主要機能にネットワーク接続を必要としません。音声ガイドは Apple の端末内 AVSpeechSynthesizer を使用します。
```

### App Review Notes - 推奨英語版

```text
MalaWatch does not require an account or login.

This build does not offer In-App Purchases. All current features, including Apple Watch support and bead styles, are included without purchase.

The app is an offline mala / prayer bead counter for iPhone and Apple Watch. Core flows: launch the app, swipe down on the 3D bead wheel to count, follow the Om Mani Padme Hum syllable guide, change the bead goal between 27/54/108, choose a bead style, switch voice mode between Follow and Silent, and reset the current round.

MalaWatch does not collect user data, does not use third-party SDKs, and does not require network access for core functionality. Voice guidance uses Apple's on-device AVSpeechSynthesizer.
```

## 提出前チェック

- App Store Connect のバージョンページに In-App Purchase を付けない。
- 説明、スクリーンショット、Review Notes に Premium / Unlock / paid Apple Watch を書かない。
- Apple Watch は included feature として説明する。
- Apple から実機録画を求められた場合、App 起動から始め、メイン画面のカウント、六音節ガイド、設定、27/54/108、珠スタイル、Follow/Silent、リセット、可能なら Apple Watch を見せる。
